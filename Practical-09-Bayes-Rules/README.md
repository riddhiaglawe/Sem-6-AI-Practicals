# Practical 09 – Bayes Rule

## Aim
To evaluate Bayes Rule for predicting rain from a given scenario.

## Objective
- Understand Bayes Theorem
- Apply conditional probability
- Solve real-world problems using probability

## Theory
Bayes Theorem is used to calculate conditional probability:

P(A|B) = [P(B|A) × P(A)] / P(B)

It helps find the probability of an event based on prior knowledge.

## Example

Given:
P(cloudy) = 0.40  
P(rain) = 0.20  
P(cloudy | rain) = 0.85  

Then:
P(rain | cloudy) = 0.20 × 0.85 / 0.40 = 0.425

So, probability of rain = **42.5%**

2. Apply Bayes formula
3. Calculate result
4. Display output

## Conclusion
Bayes Theorem helps in predicting outcomes using prior probabilities and is widely used in AI and decision-making systems.

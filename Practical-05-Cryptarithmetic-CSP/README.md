# Practical 05 – Cryptarithmetic Puzzle using CSP

## Aim
To implement Cryptarithmetic Puzzle using Constraint Satisfaction Problem (CSP) technique.

## Objective
- To understand CSP representation.
- To implement backtracking algorithm.
- To solve SEND + MORE = MONEY puzzle.

## Concepts Used
- Variables
- Domains (0-9 digits)
- Constraints (unique digit assignment)
- Backtracking
- Constraint propagation

## Example
SEND + MORE = MONEY

Solution:
9567 + 1085 = 10652

## Algorithm
1. Identify unique characters.
2. Assign digits using backtracking.
3. Ensure constraints are satisfied.
4. Stop when valid solution found.

## Conclusion
The Cryptarithmetic puzzle was successfully solved using backtracking and constraint satisfaction approach.

# Practical 08 – Semantic Networks

## Aim
To demonstrate the concept of semantic networks for knowledge representation.

## Objective
- Understand semantic networks
- Represent knowledge using nodes and relationships
- Learn ISA and HAS-A relations
- Understand inheritance in networks

## Theory
A semantic network is a graphical representation of knowledge using:
- Nodes → objects or concepts  
- Edges → relationships  

Common relations:
- ISA (is-a)
- HAS-A (property)
- CAN (ability)

It is closely related to predicate logic representation.

## Example

Dog ISA Animal  
Animal HAS cells  

Inference:
Dog HAS cells (by inheritance)

## Algorithm
1. Define nodes (objects)
2. Define relationships (ISA, HAS)
3. Store relations
4. Display network
5. Apply inheritance rules

## Conclusion
Semantic networks help represent knowledge in a structured and visual way, making reasoning and inference easier.

# Practical 03 – Tic Tac Toe Game with AI Player

## Aim
To build a Tic-Tac-Toe game engine with an AI player in Python.

## Objective
- To understand game theory basics.
- To implement decision-making logic.
- To simulate human vs computer gameplay.

## Concepts Used
- Game Tree
- Basic AI Logic
- Winning Condition Checking
- State Evaluation

## Algorithm
1. Initialize 3×3 board.
2. Accept user input.
3. AI selects optimal move.
4. Check for win or draw.
5. Repeat until game ends.

## Features
- Human vs Computer
- Win detection
- Draw detection
- Simple AI decision logic

## Conclusion
The Tic-Tac-Toe game was implemented successfully with basic AI logic and rule-based move evaluation.

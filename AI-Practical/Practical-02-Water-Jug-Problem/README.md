# Practical 02 – Water Jug Problem using Production Rules & BFS

## Aim
To apply production rules to solve the Water Jug Problem and obtain exactly 2 gallons in a 4-gallon jug.

## Objective
- To understand state-space representation.
- To implement BFS approach.
- To apply production rules in AI problems.

## Problem Statement
Two jugs:
- 4-gallon jug
- 3-gallon jug

Goal: Measure exactly 2 gallons in 4-gallon jug.

## Techniques Used
- Production Rules
- Breadth First Search (BFS)
- State-space representation

## Algorithm
1. Represent state as (x, y)
2. Define possible actions:
   - Fill jug
   - Empty jug
   - Transfer water
3. Use BFS to explore states
4. Stop when target state is reached

## Time Complexity
O(n × m)

## Conclusion
The Water Jug Problem was successfully solved using BFS and production rules to reach the goal state.

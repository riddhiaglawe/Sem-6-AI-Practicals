# Practical 10 – Prompt Engineering for Adversarial Search

## Aim
To apply prompt engineering techniques to design an effective prompt for learning the concept of Adversarial Search.

## Objective
- Understand the concept of prompts and prompt engineering  
- Learn how to design structured and effective prompts  
- Differentiate between weak and strong prompts  
- Improve output quality using prompt refinement  

## Theory
Prompt engineering is the process of designing inputs (prompts) to get accurate and meaningful responses from AI systems.

A good prompt should include:
- Clarity  
- Specificity  
- Context  
- Constraints  
- Clear intent  

Techniques used:
- Role-based prompting  
- Step-by-step prompting  
- Few-shot prompting  
- Instruction with constraints  
- Iterative refinement  

## Example

Weak Prompt:  
Explain AI  

Improved Prompt:  
Explain AI in simple terms with examples  

Refined Prompt:  
Explain Artificial Intelligence with definition, examples, and applications in 5 bullet points  

## Algorithm
1. Start with a basic prompt  
2. Analyze output quality  
3. Add clarity and constraints  
4. Refine the prompt step-by-step  
5. Evaluate final output  

## Conclusion
Prompt engineering helps in generating accurate and structured outputs from AI systems. By refining prompts, better results can be achieved efficiently.
